    • I- Introduction

    L’objectif de ce projet est de réaliser un site dynamique en utilisant l’ensemble des langages qu’on a eu à développer au cours de cette formation. Dans ce cas, nous allons vous faire un résumé de notre site de logements nommé logement à tours qui est un site ouvert tout le monde. Dans ce résumé nous allons présenter les langages utilisés pour sa réalisation, le site web en tant que tel et en fin dégager des perspectives pour le site. II- Le site et ses fonctionnalités Dans cette partie nous allons de notre site de logement en commençant d’abord par la structure de la base de données, la structure du site et son fonctionnement et en fin de l’hébergement du site.

    4- La base de données

    Figure1 : Modèle conceptuel de données (MCD) de la BD sitweb.

    Ci-dessus le MCD de la base de données nommée sitweb qui est la base avec laquelle on a travaillé pour ce projet. Elle est constituée de quatre tables qui comportent chacune des attributs qui sont en parfaits correspondance avec les informations qu’on veux avoir ou faire connaitre à nos visiteurs. Elles s’agissent : D’une table logements qui gèrent tous nos publications de logements avec les informations telles que le type de logement, la surface, le loyer et le lieu, D’une table agences qui comportent les informations sur nos agences telles que leurs lieux, les contacts et sont au nombre trois, D’une table locataires et propriétaires qui comportent eux aussi les informations dont on aura besoin pour bien communiquer avec nos locataires ou les propriétaires des logements. Et en bas, on le MLD du MCD de la base de données.

    5- La structure et fonctionnement du site

    Le site est constitué d’une page d’accueil sur laquelle on peut voir des rubriques comme tours qui indique la localisation de la ville de tours avec ses coordonnées administratives et géographiques, une barre qui comporte les rubriques des quatre tables de notre base de données. Pour les rubriques locataires et propriétaires, on trouve des formulaires avec lesquels les locataires et les propriétaires peuvent s’inscrire sur la base de donnée, une rubrique agences qui comporte les éléments de la table agences que les visiteurs peuvent prendre pour nous contacter soit par téléphone ou soit prendre l’adresse pour se rendre aux agences, et cette page comporte aussi un lien d’envoi de mail et un formulaire similaire aux formulaires de contact pour récupérer les informations de locataires ou des propriétaires. Une rubrique logements qui comportent nos annonces de logements. Dans cette page, on peut faire des recherches sur les types de logements et comporte un lien d’envoi de mail avec un formulaire similaire au formulaire de contact pour récupérer les informations sur les locataires, et sur chaque page, on a un fond d’écran qui est une image qui donne directement une idée sur le site et en bas on a un bouton dans lequel on trouve tous les moyens à travers lesquels on peut nous contacter.

    6-L’hébergement

    Une fois le travail terminé, on doit le mettre à la disposition des internautes à travers un hébergeur qui donne aussi l'accès à tous les internautes au contenu déposé dans leurs comptes par les webmestres souvent via un logiciel FTP ou un gestionnaire de fichiers. Il y en des hébergeurs payants et d'autres qui sont gratuits ou payant avec des options gratuites.     • III- Les outils

    1- HTML

    Le HTML (HyperText Mark-Up Language) est un langage de marquage, de structuration ou encore un langage de balise ayant pour rôle la formalisation de l’écriture d’un document avec des balise. Les balises permettent de montrer la manière dont on veut présenter le document et les liens avec d’autres documents. Ce langage est sous plusieurs version et permet aussi la lecture des documents sur internet à partir de différentes machines grâce à un protocole appelé http qui permet l’accès à travers le réseau à des documents repérés par une adresse unique appelée URL.

    2- CSS

    Le terme CSS (« Cascading Style Sheets ») qui peut se traduire par « feuilles de style en cascade ». C’est un langage informatique utilisé sur internet pour la mise en forme des fichiers HTML ou XML. Les fichiers CSS contiennent du code qui permet de gérer le design d’une page HTML. Bien que HTML fasse la mise en forme à l’aide des balises prévus à cet effet, le CSS permet de modifier tous les titres du fichier html à savoir les couleurs, les tailles, positions du texte en des codes qui lui sont spécifiques. Un fichier CSS peut aussi être dans un fichier HTML.

    3- PHP

    C’est un langage de scripts généraliste et open source qui est spécialement fait pour le développement d’application web et il est facile à intégrer au html. C’est un langage qui supporte les bases de données en permettant son appel en utilisant des extensions spécifiques aux bases de données (MySQL) ou PDO qui est une classe d’abstraction. Il est utilisé sur la majorité de nombreux systèmes de d’exploitation (Microsoft Windows, MacOs, RISC et d’autres encore). Il possède aussi des fonctionnalités très utiles dans le traitement de texte, des extensions pour accéder aux fichiers XML.

    1- MySQL

    Le terme MySQL « My Structured Query Language » désigne un serveur de base de données distribué sous licence libre GNU (General Public Licence). Il est la plupart intégré dans le logiciel LAMP (XAMP dans notre cas) qui comprend un système d’exploitation (Lunix), un serveur web (Apache) et un langage de scripts (PHP). En pratique, le serveur de MySQL peut être résumé à un lieu de stockage et d’enregistrement des données que se soient cryptées ou non. On peut récupérer via une requête SQL les informations sur ce serveur très rapidement. C'est le cas, par exemple, avec les mots de passe enregistrés sur des sites web. Si le serveur détecte la présence du mot de passe entré dans un formulaire dans ses données, il autorise la connexion. S'il ne trouve pas le mot de passe, la connexion sera refusée. Il possède des interfaces dont PHPMyAdmin qui permet de gérer la base de données sur un serveur PHP. Cette interface pratique permet d'exécuter, très facilement et sans grandes connaissances en bases de données, des requêtes comme les créations de table de données, insertions, mises à jour, suppressions et modifications de structure de la base de données, ainsi que l'attribution et la révocation de droits et l'import/export. Ce système permet de sauvegarder commodément une base de données sous forme de fichier .sql et d'y transférer ses données, même sans connaître SQL. l s'agit de l'une des plus célèbres interfaces pour gérer une base de données MySQL sur un serveur PHP.
